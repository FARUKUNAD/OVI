Objetivo General


Realizar un Objeto Virtual de Aprendizaje (OVI) a través de plataformas tecnológicas basadas en HTML 5 Y CSS3 cuyo fin es adquirir e interiorizar los conocimientos y destrezas en la creación de sitios web.


GIT Y GITHUB GESTORES DE PROYECTOS COLABORATIVOS DE DISEÑO WEB.



Dar a conocer por medio del Lenguaje HTML5 los conceptos y las bases necesarias para el manejo de Git y GitHub, con el propósito que desarrolladores la conozcan y usen para poder llevar un control más efectivo de las versiones, cambios, mejoras de proyecto de desarrollo de software.


MAQUETACIÓN O MOCKUP.




Dar a conocer por medio del Lenguaje HTML5 el concepto de maquetación o Mockup de la página de un sitio web, además alguna de las herramientas de creación de Mockups existentes en la actualidad, que permitirán al desarrollador contar con un primer diseño que guía el proyecto, este diseño será la representación en baja calidad que mostrará los elementos de una web y su colocación.

HTML5 y CSS3 PARA CREACIÓN DE CONTENIDOS EN FORMATO WEB (Páginas y Sitios Web).


Dar a conocer los conceptos y las bases necesarias para el manejo de HTML 5 y CSS, mostrar lo que es un elemento o etiqueta HTML, lo que significa un atributo, la sintaxis básica de marcación HTML, la estructura general de una página Web, y demás. En términos de CSS, se esperará familiarizar con los selectores basados en elemento, clase e ID, con la sintaxis de una propiedad CSS y con cómo incluir CSS en sus páginas Web utilizando hojas de estilo incorporadas o externas.
